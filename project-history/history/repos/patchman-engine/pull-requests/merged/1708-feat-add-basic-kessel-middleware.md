Tests:

## Reviewer's Guide

This PR adds a new Gin middleware that integrates with the Kessel RBAC service: it conditionally attaches to protected routes based on configuration, streams workspace permissions over gRPC, processes them into inventory groups, and stores them in the request context. It also introduces feature flags, updates routes and module dependencies, and provides unit tests for the Kessel middleware.

#### Sequence diagram for Kessel middleware permission check

```mermaid
sequenceDiagram
    participant Client as actor Client
    participant Gin as Gin Middleware
    participant Kessel as Kessel RBAC Service
    participant Utils as Utils

    Client->>Gin: HTTP request (protected route)
    Gin->>Utils: Parse x-rh-identity header
    alt Invalid x-rh-identity
        Gin-->>Client: 401 Unauthorized
    else Valid x-rh-identity
        Gin->>Kessel: StreamedListObjects gRPC request
        alt gRPC error
            Gin-->>Client: 500 Internal Server Error
        else gRPC success
            Kessel-->>Gin: Streamed workspace permissions
            Gin->>Utils: Process workspace responses
            alt No workspaces found
                Gin-->>Client: 401 Unauthorized
            else Workspaces found
                Gin->>Gin: Set inventory groups in context
                Gin-->>Client: Continue to next handler
            end
        end
    end
```
