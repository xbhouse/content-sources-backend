Remove the legacy inventory schema and Cyndi integration, replacing them with direct system inventory metrics and aligning database migrations, configuration, and monitoring with the new approach.

New Features:
- Expose new Prometheus metrics for system inventory counts and tag usage, replacing previous Cyndi-based metrics.

Enhancements:
- Remove dependency on the inventory schema, roles, and Cyndi integration from database schema, configuration, and test data.
- Simplify inventory-related filtering logic by dropping feature flags tied to Cyndi tags.
- Adjust database tests, fixtures, and helper scripts to work without the inventory schema while keeping migrations compatible.
- Update Grafana dashboards to consume the new system inventory metrics instead of Cyndi metrics.

Deployment:
- Clean up deployment configuration by removing Cyndi-specific passwords, configuration, and integration blocks.

Tests:
- Add database-backed tests for the new system inventory metrics and update existing tests to reflect the removal of the inventory schema.
