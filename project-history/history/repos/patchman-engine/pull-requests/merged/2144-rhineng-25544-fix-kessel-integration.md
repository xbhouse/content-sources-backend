## Description

## Summary by Sourcery

Extract user IDs directly from x-rh-identity headers for Kessel authorization and adjust middleware/tests accordingly.

New Features:
- Add utility to retrieve user_id from encoded x-rh-identity headers using fast JSON parsing.

Bug Fixes:
- Ensure Kessel permission checks use the user_id from x-rh-identity, falling back to service_account when necessary, and return unauthorized when no user_id is present.

Enhancements:
- Refine Kessel middleware to depend on user_id strings rather than full identity objects.
- Expand identity and Kessel middleware tests to cover user_id extraction paths and missing-user_id error handling.

Build:
- Add fastjson dependency for efficient identity header parsing.
