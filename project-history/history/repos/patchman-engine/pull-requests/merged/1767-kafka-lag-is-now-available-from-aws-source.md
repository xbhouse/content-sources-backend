## Summary by Sourcery

Update the Patchman Engine Grafana dashboard to source Kafka lag metrics from AWS, modernize panel configurations, streamline datasource handling via variables, and adopt consistent naming patterns for stage/production clusters.

New Features:
- Add Kafka consumer lag panel sourcing metrics from AWS (aws_kafka_sum_offset_lag_sum)

Enhancements:
- Replace hardcoded Prometheus datasources with variable-driven references for flexibility
- Bump all panel plugin versions to 11.6.3 and update schemaVersion to 41
- Standardize panel settings by adding percentChangeColorMode, barWidthFactor, hideZeros, and remove null step values
- Enable dynamic cluster naming in variables using *-stage and *-production regex patterns
- Remove redundant config entries (liveNow, hideUrlSync flags, default null values, legacy timepicker settings)
- Set preload to false for dashboard performance optimization
