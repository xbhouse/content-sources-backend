This [commit](https://github.com/RedHatInsights/patchman-engine/commit/2e48130caeb289c18d4fd95ec1371709003b55aa) changed `OwnerID` to a pointer. This adds a guard for that in `/listener/upload.go` to prevent nil pointer dereferences.
