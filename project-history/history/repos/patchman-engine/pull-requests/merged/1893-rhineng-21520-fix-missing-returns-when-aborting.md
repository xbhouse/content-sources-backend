# Pull Request #1893: RHINENG-21520: fix missing returns when aborting

**Author**: @Dugowitch
**Created**: October 24, 2025 at 01:14 PM UTC
**Status**: Merged
**Labels**: None
**Base**: `master` ← **Head**: `fix-returns`

## Description

Missing returns could cause multiple abort calls, which would rewrite the status code and the error message, thus providing an incorrect response, and trigger excessive error logs.
