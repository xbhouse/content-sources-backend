## Summary by Sourcery

Add regression tests to ensure notifications are not sent when there are no new advisories to notify.

Tests:
- Add tests verifying no notification is produced when all advisories for a system have already been notified.
- Add a test ensuring no notification is produced when publishNewAdvisoriesNotification is called with an empty advisory map.
- Add a test documenting that getUnnotifiedAdvisories returns a non-nil empty slice when there are no unnotified advisories.
