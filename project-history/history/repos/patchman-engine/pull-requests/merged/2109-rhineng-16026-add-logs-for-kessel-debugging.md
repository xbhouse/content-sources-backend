<issue_to_address>
**issue (bug_risk):** Deferred LogDebug call will log a near-zero duration because arguments are evaluated at defer time, not at execution time.

To capture the real duration, you can wrap the log in an anonymous function so `time.Since` is evaluated at the end, e.g.:

```go
defer func(start time.Time) {
    utils.LogDebug("durationMs", time.Since(start).Milliseconds(), "processed workspaces")
}(time.Now())
```

Alternatively, compute `elapsed := time.Since(start).Milliseconds()` just before returning and log it there.
