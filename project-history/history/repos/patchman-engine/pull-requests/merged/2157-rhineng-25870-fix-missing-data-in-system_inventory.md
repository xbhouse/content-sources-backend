## Description

This PR:
- updates the list of columns that should be updated to include the latest additions for the system_inventory
- updates tests to check system for all fields that should be updated

## Summary by Sourcery

Ensure system inventory profile fields are refreshed when updating an existing system platform record.

Bug Fixes:
- Fix system_inventory updates so host-derived profile columns are refreshed on ON CONFLICT DO UPDATE, not only on first insert.

Enhancements:
- Add coverage helper to compare system_inventory profile fields against host data for consistency.

Tests:
- Extend upload and platform update tests to verify all relevant system_inventory profile fields are correctly refreshed on subsequent updates.
