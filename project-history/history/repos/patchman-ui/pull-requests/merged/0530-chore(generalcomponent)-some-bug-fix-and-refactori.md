## Description

Better logic and some bug fix. Nikhil reported a bug related to an unwanted error notification on an unauthorized user accesses Patch. But I was not able to disable automatic notification which is being generated by  [notificationsMiddleware](https://github.com/RedHatInsights/patchman-ui/blob/master/src/store/index.js#L2). This automatic error notification can be disabled [completely or for a single action](https://github.com/RedHatInsights/frontend-components/blob/master/packages/notifications/doc/notifications.md). But both options do not work to the current version of the UI. Later on, we can disable automatic error notifications and handle them on our own later on.
